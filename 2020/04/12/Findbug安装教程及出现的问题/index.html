Findbug安装教程及出现的问题

1.使用eclipse安装：

  • Help –> Install New Software –>Add
    • Name:findbugs
    • Location:http://findbugs.cs.umd.edu/eclipse
  • Add–>finish

2.安装出现的问题：

  • 下载速度慢，常卡在48%
    • PC端使用翻墙软件,重新安装findbug
    • 手机使用翻墙软件，分享热点，重新安装findbug

测试：

操作：右键点击.class文件，点击findbug

若无 Bug Explorer: Window –>show view –>other—>选择Bug Explorer

若Bug Explorer 无内容：

  • 本文链接： http://yoursite.com/2020/04/12/Findbug%E5%AE%89%E8%A3%85%E6%95%99%E7%A8%8B%E5%8F%8A%E5%87%BA%E7%8E%B0%E7%9A%84%E9%97%AE%E9%A2%98/
