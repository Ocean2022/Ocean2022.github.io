WebDriver API

  • WebDriver API-多窗口切换

    在页面操作过程中可能会出现一个新的窗口，需要切换到新的窗口进行操作

    方法：

    • driver.current_window_handle:获得当前窗口句柄
    • driver.window_handles:返回所以窗口的句柄到当前会话
    • driver.switch_to.window(handle):切换到相应的窗口，在不同窗口之间进行切换

  • WebDriver API-处理提示框

  • 警告框的处理

    • WebDriver中处理JS生成的alert、confirm和prompt
    • driver.swicth_to.aleert:定位到提示窗口（旧：driver.swich_to_alert())
    • 然后使用text/accept/dismiss/send_keys等方法进行操作
    • text:返回 alert/confirm/prompt中的文字信息
    • accept():点击确认，接受警告框
    • dissmiss()：点击取消或者“x”，解散现有警告框
    • send_key(keys):发送文本至警告框，仅限于prompr,在alter和confirm上没有输入框
    • 警示框出现的步骤肯定实在某一个时间之后的，但你的场景也可能是不一样出现警示框
    • 所以要用try except去捕捉警示框，如果警示框出现，就切换至警示框进行操作了。
